♻️ AI Smart Waste Collection System

📌 Overview
The AI Smart Waste Collection System is a web-based application built using Python and Streamlit. It helps users identify waste types, learn proper waste segregation methods, and provides recycling guidance through an interactive AI chatbot. The goal of this project is to promote environmental awareness and sustainable waste management.

🚀 Features
AI-powered chatbot for waste-related queries
Waste segregation guidance (wet, dry, recyclable, hazardous)
Recycling information and eco-friendly tips
Simple and user-friendly interface
Real-time responses for user questions

🛠️ Technologies Used
Python
Streamlit
AI-based Chatbot
